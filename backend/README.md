# Customer Portal Backend

## Setting up / Running

```shell
yarn
yarn start
```

or

```shell
npm install
npm start
``` 

Backend can be accessed at `http://localhost:9000`

## API Endpoints

* **Users** 
    * `GET` `/users`
    * `POST` `/users`
* **Accounts** 
    * `GET` `/accounts` 
    
## Additional notes

Due to the nature of this app, backend authentication is not required at this time.
